## Description

This GitHub Repository is meant to be a companion for the Master Thesis "Between Real and Unreal: A Qualitative Analysis of User Responses to Qatar Airways’ Virtual Cabin Crew on Instagram" by Tanja Altenburger (2025). Please refer to the paper for more detailed information.


**If you clone this repository and run the code by yourself, make sure to:**

- Use the following file to open the project: `R Analysis Master Thesis.Rproj`

## How To Use

There are two scripts available, one that analyzes Instagram comments (`Thematic Content Analysis Instagram Comments.R`) and one that analyzes the responses of guided interviews (`Qualitative Analysis Interviews.R`). Please open the respective script depending on what data you would like to analyze. Furthermore, you can find the Excel files with the input data in the folder `Data` as well as all the graphs generated through this code in the folder `Output`.
